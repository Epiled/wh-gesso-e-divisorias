## 📦 Install and Use

**Prerequisites:** Node.js (v22.x) or higher installed.

1. Clone the repository:
```bash
git clone https://github.com/Epiled/wh-gesso-e-divisorias.git
cd wh-gesso-e-divisorias
```

2. Install the dependencies:
```bash
npm install
```

3. Run the development environment (Build + Watch + Server):
```bash
npm run dev
```

4. (Optional) Generate minified build for production:
```bash
npm run build
```
